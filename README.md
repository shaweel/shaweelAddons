# shaweelAddons
The goofiest SkyBlock mod out there with features never seen before like:
## Features
- (1.8.9 only) Sea Creature Alert: Notifies you whenever you catch a sea creature
- (1.8.9 only) SS Reset Alert: Notifies you when someone says that ss reset in a dungeon
- (1.8.9 only) Low Health: Notifies you when you have low health
- (1.8.9 only) Class Milestone Reminder: Shows a title before you get milestone 3, useful for archers using rcm in clear
- (1.8.9 only) Katana Hud: Shows whether the Enderman Slayer's katana weapon has it's ability activated
- (1.8.9 only) Katana Ability Expired Sound - Plays a sound whenever your Katana ability expires
- Shitter List - You can add people to your shitter list, which will auto kick them with reason. /sha shitterlist
- (1.8.9 only) Blow Gate Alert - Alerts you whenever you need to blow the gate in F7 terminals based on what you did
- (1.8.9 only) Extra Life Items Alert - Alerts you whenever your Bonzo or Spirit mask pops, or whenever your Phoenix dies
- (1.8.9 only) Dungeon Chest Profit - Calculates profit in dungeon chests(cuz NEU eats fps for breakfast, lunch and dinner)
- (1.8.9 only) Splits - Dungeon splits with the most information out of any mod.
- (1.8.9 only) Mimic Killed - Alerts you whenever Mimic is killed in a Dungeon
- (1.8.9 only) Rat Kill Alert - Alerts you whenever you kill a hub Rat
- (1.8.9 only) Leap overlay - A leap overlay with keybinds
- (1.8.9 only) Wish alert - Tells you when to wish in dungeons
## Install
- This should work on every operating system, tested on: Windows, MacOS, Linux(Arch Linux)
1. Download the file called *shaweelAddons-x.x.x.jar* from the ![latest release](https://github.com/shaweel/shaweelAddons/releases/latest)
2. Download and install [Fabric](https://fabricmc.net/use/installer/) for *Minecraft 1.21.10* if you don't have it already
4. Drag the *shaweelAddons-x.x.x.jar* into the mods folder inside your .minecraft folder
